# Changelog

All notable changes to this project will be documented here.

## Unreleased

- Initial open-source release of the MiniMax SAPI5 TTS bridge.
- WinForms TTS manager with provider selection, voice presets, custom voice ids, local service controls, and SAPI5 registration launcher.
- Native SAPI5 HTTP bridge (x64, raw C++/COM, no ATL).
- PowerShell voice registration script supporting both HKCU and HKLM.
- MiniMax native T2A support and OpenAI-compatible TTS support in the manager.
- Optional Python local server for Edge TTS or MiniMax proxy experiments.
- Replaced placeholder GUIDs, personal paths, and `YOUR_NAME` references with project values.
- Removed legacy Zotero-plugin experiment (`src/`, `addon/`, `build/`); the SAPI5 bridge is the supported path.
